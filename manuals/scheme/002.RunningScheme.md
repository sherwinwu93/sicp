## 2.3 Memory Usage 内存使用
1. stack: 栈
2. heap: 堆
3. constant space: 常量区域
5. extra storage: 额外存储

内存总量 = stack+(constant + 2*heap) + extra

当单heap时 stack+(constant+ heap) + extra

(print-gc-statistics) 展示内存使用情况
