** 2.3 Memory Usage 内存使用
   能用多大内存,内存怎么分配的
   Scheme使用4种内容:
   1. stack: 递归过程
   2. heap: 动态对象
   3. constant space: 静态对象,生命周期和scheme一致
   4. extra storage: 微内核
   scheme有两种scheme: constant和heap共用内存
   1. scheme: 2个heap gc ->stack + (constant + 2*heap) + extra
   2. bchscheme: 1个heap gc , 一个disk file gc ->stack + (constant + heap) + extra
